Kurulum #

IconPicker'ı kullanmak için aşağıdaki adımları takip edin:

1. Gerekli CSS ve JS Dosyalarını Ekleyin

Öncelikle Bootstrap, Font Awesome ve IconPicker'ın CSS ve JS dosyalarını HTML sayfanıza ekleyin:

<!-- CSS Dosyaları -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.4/css/all.min.css" rel="stylesheet">
<link href="css/iconpicker.css" rel="stylesheet">

<!-- JavaScript Dosyaları -->
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
<script src="js/iconset/iconset-fontawesome-5-all.js"></script>
<script src="js/iconpicker.js"></script>

Not

Unutmayın ki IconPicker, jQuery'nin yüklü olmasını gerektirir. Ayrıca, Font Awesome ikon seti için iconset-fontawesome-5-all.js dosyasını da eklemeniz gerekmektedir.

Temel Kullanım #

IconPicker'ı bir input alanı için etkinleştirmek basittir:

HTML Yapısı

<div class="input-group">
    <div class="preview-icon">
        <i class="fas fa-heart"></i>
    </div>
    <input type="text" class="form-control iconpicker-basic" name="icon" value="fas fa-heart">
</div>

JavaScript Kodu

$(document).ready(function() {
    $('.iconpicker-basic').iconpicker();
});

Ayarlar #

IconPicker birçok özelleştirme seçeneği sunar:

$('.iconpicker').iconpicker({
    title: 'İkon Seçici',             // Başlık
    selectedIcon: 'fas fa-heart',     // Varsayılan seçili ikon
    defaultCategory: 'solid',         // Varsayılan kategori
    height: 280,                      // Picker yüksekliği (px)
    hideOnSelect: true,               // Seçimden sonra gizle
    showFooter: true,                 // Alt bilgi göster/gizle
    iconPreview: '.preview-icon i',   // Önizleme elementi seçicisi
    searchText: 'İkon Ara...',        // Arama kutucuğu metni
    noResultsText: 'Sonuç bulunamadı',// Sonuç bulunamadığında gösterilecek metin
    borderRadius: '.375rem',          // Köşe yuvarlaklığı
    backdropOpacity: 0.7,             // Arkaplan opaklığı
    placement: 'bottom',              // Yerleşim (top/bottom)
    modalMode: false,                 // Modal içinde kullanım
    zIndex: 1040,                     // z-index değeri
    inlineMode: false                 // Inline mod
});
Seçenek Tür Varsayılan Açıklama
title String 'İkon Seçici' Picker başlığı
selectedIcon String 'fas fa-heart' Varsayılan seçili ikon
defaultCategory String 'solid' Varsayılan kategori
height Number 280 İkon listesinin maksimum yüksekliği (px)
hideOnSelect Boolean true İkon seçildikten sonra dropdown'ı gizle
showFooter Boolean true Alt bilgiyi göster/gizle
iconPreview String/jQuery null İkon önizleme elementinin seçicisi
modalMode Boolean false Modal içinde kullanım için optimizasyon
inlineMode Boolean false Inline gösterim modu

Modal İçinde Kullanım #

Modal içinde kullanıldığında, modalMode: true ayarı ile daha iyi bir deneyim sunabilirsiniz:

Modal İçinde IconPicker

HTML Kodu

<!-- Button trigger modal -->
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#iconPickerModal">
    <i class="fas fa-icons me-2"></i> Modal İçinde İkon Seçin
</button>

<!-- Modal -->
<div class="modal fade" id="iconPickerModal" tabindex="-1" aria-labelledby="iconPickerModalLabel" aria-hidden="true">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">
                <h5 class="modal-title" id="iconPickerModalLabel">Modal İçinde IconPicker</h5>
                <button type="button" class="btn-close" data-bs-dismiss="modal" aria-label="Close"></button>
            </div>
            <div class="modal-body">
                <div class="input-group">
                    <div class="preview-icon">
                        <i class="fas fa-star"></i>
                    </div>
                    <input type="text" class="form-control iconpicker-modal" name="icon" value="fas fa-star">
                </div>
            </div>
            <div class="modal-footer">
                <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Kapat</button>
                <button type="button" class="btn btn-primary">Kaydet</button>
            </div>
        </div>
    </div>
</div>

JavaScript

$('.iconpicker-modal').iconpicker({
    iconPreview: '.preview-icon i',
    modalMode: true
});

Callback Fonksiyonları #

Icon seçimi sırasında özel olayları dinleyebilirsiniz:

İkon seçimi bekleniyor...

HTML Kodu

<div class="input-group">
    <div class="preview-icon">
        <i class="fas fa-cog"></i>
    </div>
    <input type="text" class="form-control iconpicker-callback" name="icon" value="fas fa-cog">
</div>
<div class="alert alert-info mt-3" id="callbackResult">İkon seçimi bekleniyor...</div>

JavaScript

$('.iconpicker-callback').iconpicker({
    iconPreview: '.preview-icon i'
}).on('change', function(e) {
    $('#callbackResult').html(
        '<strong>İkon değiştirildi:</strong> ' + 
        '<i class="' + e.target.value + ' me-2"></i>' + 
        '<code>' + e.target.value + '</code>'
    );
});

Metodlar #

İşte IconPicker'ın sunduğu bazı kullanışlı metodlar:

// Dropdown'ı göster
$('.iconpicker').data('iconpicker').showDropdown();

// Dropdown'ı gizle
$('.iconpicker').data('iconpicker').hideDropdown();

// İkon seç
$('.iconpicker').data('iconpicker').selectIcon('fas fa-check');

// Programsal olarak ikonu değiştir (EN BASİT YÖNTEM)
$('.iconpicker').data('iconpicker').setIcon('fas fa-star');

// Veya pratik bir yardımcı fonksiyon ile:
function setIconValue(selector, iconClass) {
    var iconpicker = $(selector).data('iconpicker');
    if (iconpicker && typeof iconpicker.setIcon === 'function') {
        return iconpicker.setIcon(iconClass);
    }
    return false;
}
// Kullanımı: setIconValue('#myIconPicker', 'fas fa-rocket');
Metod Açıklama
showDropdown() IconPicker dropdown'ını gösterir.
hideDropdown() IconPicker dropdown'ını gizler.
selectIcon(iconClass) Programatik olarak bir ikon seçer.
setIcon(iconClass) İkonu programsal olarak değiştirir ve önizlemeyi günceller. En basit ve etkili yöntemdir.
updateSettings(settings) Picker ayarlarını günceller.

Olaylar #

IconPicker çeşitli olaylar tetikler:

$('.iconpicker')
    .on('shown.iconpicker', function(e) {
        console.log('IconPicker açıldı');
    })
    .on('hidden.iconpicker', function(e) {
        console.log('IconPicker kapandı');
    })
    .on('selected.iconpicker', function(e, iconClass) {
        console.log('Seçilen ikon:', iconClass);
    })
    .on('change', function(e) {
        console.log('Input değeri değişti:', e.target.value);
    });
Olay Açıklama
shown.iconpicker Dropdown gösterildiğinde tetiklenir.
hidden.iconpicker Dropdown gizlendiğinde tetiklenir.
selected.iconpicker Bir ikon seçildiğinde tetiklenir.
change Input değeri değiştiğinde tetiklenir (standart input olayı).
