jQuery Icon Picker

Font Awesome 5 ve Bootstrap 5.3.3 Uyumlu
Demolar
1. Temel Kullanım
Seçilen: fas fa-heart
2. Sadece Buton
Seçilen: fas fa-star
3. Form İçinde
Kullanım
1. Gerekli dosyaları ekleyin:
<!-- Bootstrap 5.3.3 CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Font Awesome 5 CSS -->
<link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.4/css/all.min.css" rel="stylesheet">

<!-- Icon Picker CSS -->
<link href="css/iconpicker.css" rel="stylesheet">

<!-- jQuery -->
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

<!-- Bootstrap 5.3.3 JS -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

<!-- Icon Picker JS -->
<script src="js/iconpicker.js"></script>
2. HTML yapısını oluşturun:
<div class="input-group">
    <span class="input-group-text"><i id="icon-preview" class="fas fa-heart"></i></span>
    <input type="text" class="form-control" id="icon-input" value="fas fa-heart">
    <button class="btn btn-outline-secondary" type="button" id="icon-button">İkon Seç</button>
</div>
3. JavaScript ile başlatın:
$(document).ready(function() {
    $('#icon-input').iconpicker({
        title: 'İkon Seçin',
        selectedIcon: 'fas fa-heart',
        iconPreview: '#icon-preview',
        iconButton: '#icon-button',
        onChange: function(iconClass) {
            console.log('Seçilen ikon: ' + iconClass);
        }
    });
});
Akıllı Dosya Yolu Tespiti: IconPicker dosya yollarını otomatik olarak tespit eder. İkonları kütüphanenin bulunduğu dizindeki iconset/iconset-fontawesome-5-all.js dosyasından otomatik olarak yükler.
Seçenekler
$('#icon-input').iconpicker({
    // Başlık
    title: 'İkon Seçin',
    
    // Başlangıç ikonu
    selectedIcon: 'fas fa-heart',
    
    // Varsayılan kategori
    defaultCategory: 'solid',
    
    // İkon önizleme elementi
    iconPreview: '#icon-preview',
    
    // İkon buton seçici
    iconButton: '#icon-button',
    
    // Seçimde kapat
    hideOnSelect: true,
    
    // Alt bilgi göster
    showFooter: true,
    
    // Arama metni
    searchText: 'İkon Ara...',
    
    // Sonuç bulunamadı metni
    noResultsText: 'Sonuç bulunamadı',
    
    // İkon seçildiğinde callback
    onChange: function(iconClass) {
        console.log('Seçilen ikon: ' + iconClass);
    }
});
Metotlar
  • showDropdown Picker'ı açar
  • hideDropdown Picker'ı kapatır
  • toggleDropdown Açık/kapalı durumunu değiştirir
