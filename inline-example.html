Inline İkon Seçici Örneği

Bu örnek, IconPicker'ı dropdown yerine inputun altında doğrudan görüntüler.

Normal Kullanım
Modal İçinde Kullanım
Kod Örneği
// Özel Ayarlar ile Başlat
$('#inline-icon-input').iconpicker({
    title: 'İnline İkon Seçin',
    selectedIcon: 'fas fa-heart',
    inlineMode: true // Bu, dropdown yerine direkt görüntüleme sağlar
});

// Veya Sadece Sınıf Kullanarak Otomatik Başlat
<input type="text" class="form-control icon-picker" value="fas fa-heart">
Otomatik Başlatma Örneği

Sadece input'a icon-picker sınıfı vererek otomatik başlatma:

<!-- Sadece bir input ile kullanım -->
<input type="text" class="form-control icon-picker" value="fas fa-star">
Modal İçinde Otomatik Başlatma
Modal İçinde İkon Seçin
Önemli: Modal içinde kullanımda otomatik yapı oluşturulur. Özel ayarlar için modalMode: true seçeneğini kullanabilirsiniz. Normal input olarak düşünün, backend'e veri gönderirken icon-picker sınıfını temizlemenize gerek yoktur.
